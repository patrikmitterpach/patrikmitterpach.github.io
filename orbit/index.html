Frequently Asked Questions (FAQ)

What is this site?

SatMonitor is a project aimed at monitoring a specific satellite currently in orbit and presenting this information to the user in an aesthetically pleasing way.

What are TLE files?

"A two-line element set (TLE) is a data format encoding a list of orbital elements of an Earth-orbiting object for a given point in time"
- Wikipedia

In short, it's a two (or three) line file that contains all the information needed to graph an orbit of an extraterrestrial object in the orbit. The files are not stable, as the spacecraft orbits change by a little bit every now and then - gravitational effects, evasive maneuvers and so. For a complete repository of all (public, non-military) satellites, visit celestrak.org.

The palette is called "The Lost Century" and is available at lospec.com. I also use it with my pixel art, available at /pixel-art.

What open-source projects did you use?

Most of the math behind this site is provided by tle.js, available at gh.com/tle.js. This project is really well written and is imported into this site fully.
